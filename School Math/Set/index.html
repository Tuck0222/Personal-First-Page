ベン図による2つの集合A,Bの表示。

  1. 集合と要素
    :集合とは要素(元)の集まり。
  2. 空集合
    :要素を一つも持たない集合。Φで表す。

    3. ここからは、異なる2つの集合A,Bを定めるとする。

  3. 部分集合
    :ある集合Aの要素のすべてが集合Bの要素であること。

    (つまり、(集合A)かつ(集合Bでない＝集合Bの補集合)部分全体が空集合になっているということ。)

  4. 共通部分
    ：ある集合Aと集合Bのどちらにも共通する要素全体の集合。A∩Bと表す。
  5. 和集合
    ：ある集合Aと集合Bの少なくともどちらか一方に属する要素全体の集合。A∪Bと表す。
  6. 補集合と普遍(全体)集合
    ：ある考察対象全体を普遍集合または全体集合と呼び、考える対象の全てを部分集合にするものとする。さらに補集合とはある集合でないもの全体の集合である。ある集合Aの補集合はĀやA'で表す。

    （集合Bと集合Bの補集合は共通部分が空集合。そしてそれらの和集合は全体集合に一致する。つまり、B∩B'=Φ　B∪B'=U）

  7. ド・モルガン則
    (A∩B)'=A'∪B'
    (A∪B)'=A'∩B'
    つまり、「AとBの(共通部分)の否定」は「A'とB'の(和集合)」になるという規則のこと。(共通部分)↔(和集合)で互いにもう一方の式になる。成り立つことをベン図から確認できる。
  8. 集合の演算法則

    • 交換法則
      A∩B=B∩A
      A∪B=B∪A
    • 結合法則
      (A∩B)∩C=A∩(B∩C)
      (A∪B)∪C=A∪(B∪C)
    • 分配法則
      A∩(B∪C)=(A∪B)∩(A∪C)
      A∪(B∩C)=(A∪B)∩(A∪C)
